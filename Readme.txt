Flutter ile geliştirmiş olduğum Hava durumu uygulamasıdır. Web 
servislerimi "http://api.weatherstack.com" adresinden çektim. 

- MVVM prensibi ile projemi derli hale getirdim.
- http kütüphanesi ile servislerimi çektim.
- Türkiyedeki tüm iller için hava durumu mevcuttur.
- Her bir hava koşuluna göre hava açıksa açık renkli tema, kapalı ise 
 kapalı tema veya yağmurlu ise ayrı renk teması kullandım.
- Projemin heryerinde OOP kullanarak projemi derli hale getirdim.
- Açılış ekranı animasyonludur.

________________________________________________________________________

It is the Weather application that I developed with Flutter. web
I pulled my services from "http://api.weathersack.com".

- I compiled my project with the MVVM principle.
- I pulled my services with the http library.
- Weather is available for all provinces in Turkey.
- Light color theme if the weather is clear, if it is closed according to each weather condition
  I used a closed theme or a separate color theme if it's rainy.
- I compile my project using OOP all over my project.
- The splash screen is animated.
